# LearnUEDemo


## 📖 项目简介

**LearnUEDemo** 是基于 **Unreal Engine 5 第一人称模板** 的个人学习项目。主要通过实践官方文档中的概念和示例来深入理解UE5游戏开发。

这个项目是我学习UE5的实践记录，重点在于：

*   **理解第一人称游戏机制**：基于官方模板进行扩展和修改
*   **实践官方文档内容**：将文档中的理论知识转化为实际代码
*   **系统学习UE5核心模块**：按照官方推荐的学习路径逐步深入
*   **问题记录与解决**：记录学习过程中遇到的问题和解决方案

### 环境要求

*   **Unreal Engine 5.6**
*   **Visual Studio 2022** 或 **Rider for Unreal**
*   **Git LFS** (用于资源文件管理)
